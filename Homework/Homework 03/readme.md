# Homework 03

### 内容如下：
用⾃⼰⼿机拍摄⼀个场景，拍摄⽅式为从左到右移动拍摄，从拍摄的图像中均匀选取五张以上图像，拼接流程按照课程所讲述的⽅式来进⾏：
- 采⽤ OpenCV 中的 `SIFT` ⽅法来进⾏特征点检测，并构建不同图像特征点之间的对应关系；
- 结合 `RANSAC` ⽅法与 `Homography Transformation` 计算⽅法，获得图像对之间的变换；
- 对图像进⾏warping与融合获得最终的⼤场景图像。
